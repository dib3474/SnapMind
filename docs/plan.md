# **프로젝트 주제**

## **SnapMind**

“왜 저장했는지 잊어버리는 스크린샷/사진을 AI 기반으로 자동 분류하고 기록하는 앱”

---

# **프로젝트 개요**

사용자가 스마트폰에서 스크린샷이나 사진을 찍은 후 공유 기능을 통해 앱에 저장하면:

- 스크린샷의 종류를 머신러닝으로 자동 분류
- OCR로 텍스트 추출
- 태그 자동 추천
- 메모 및 저장 이유 기록
- 검색 및 필터링 제공

을 수행하는 AI 기반 스크린샷/사진 기록 관리 앱.

---

# **핵심 문제 정의**

사용자는:

- 유튜브
- 강의자료
- 코드 에러
- 쇼핑
- 지도
- 링크

등을 저장하기 위해 스크린샷을 자주 찍지만,

시간이 지나면:

- 왜 저장했는지
- 어떤 내용인지
- 어디에 쓰려고 했는지

기억하지 못하는 문제가 존재함.

본 앱은 이를 해결하기 위해:

- 자동 분류
- 자동 태깅
- 검색 기능
- 메모 기록

을 제공함.

---

# **앱 핵심 기능**

## **1. 스크린샷 공유 저장 기능**

### **기능 설명**

사용자가 갤러리 또는 스크린샷 화면에서 “공유” 버튼을 눌러 앱으로 이미지를 전달.

### **사용 기술**

- Android Intent
- ACTION_SEND
- 외부 앱 연동

### **평가 항목**

- 외부 APP 연동
- Intent 데이터 전달

---

# **2. 머신러닝 기반 이미지 자동 분류**

## **기능 설명**

이미지를 다음 카테고리로 자동 분류:

- video
- code
- shopping
- text
- map
- chat
- meme

### **사용 기술**

- TensorFlow Lite
- CNN 기반 간단한 이미지 분류 모델

### **학습 방식**

직접 수집한 이미지 데이터셋 사용.

### **평가 항목**

- 머신러닝

---

# **3. OCR 텍스트 추출**

## **기능 설명**

스크린샷/사진 내 텍스트를 자동 추출하여:

- 제목 추천
- 키워드 생성
- 검색 기능

에 활용.

### **사용 기술**

- ML Kit Text Recognition

### **예시**

유튜브 스크린샷:

- 영상 제목 추출

코드 에러 화면:

- Exception 이름 추출

---

# **4. 자동 태그 추천 기능**

## **기능 설명**

OCR 결과 + 이미지 분류 결과를 기반으로:

- #과제
- #영상
- #버그
- #구매예정
- #공부

등 자동 태그 추천.

### **사용 기술**

- 간단한 텍스트 키워드 분석
- OpenAI API 활용 가능

### **API 활용 예시**

“이 스크린샷은 React 에러 해결 관련 이미지입니다.”

### **평가 항목**

- API 활용

---

- 왜 저장했는지
- 나중에 할 일
- 참고 목적

등을 간단하게 메모 가능.

### **예시**

- “리액트 에러 해결 참고”
- “다음에 살 물건”
