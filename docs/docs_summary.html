TFLite 이미지 자동 분류
직접 학습한 CNN 모델로 저장된 이미지를 chat · receipt · code · shopping · travel · food · document 등 카테고리로 자동 분류합니다. Confidence Score와 함께 Room DB에 저장됩니다.

✨
Gemini API — 메모 자동 추천
이미지 저장 시 Gemini API가 이미지 내용을 분석하여 "왜 저장했는지"에 대한 메모 문장을 자동 추천합니다. 사용자는 추천 메모를 수락하거나 직접 수정할 수 있습니다.

Google Cloud Vision API — 이미지 라벨링
Vision API로 이미지 내 객체·장면·텍스트를 정밀 분석하여 자동 태그를 생성합니다. ML Kit OCR과 함께 동작하여 텍스트 기반 태그와 시각 기반 태그를 동시에 제공합니다.

YouTube Data API — 영상 바로 이동

갤러리 공유 연동 (외부 APP 연동)
Android ACTION_SEND Intent를 통해 갤러리·카카오톡·브라우저 등 외부 앱에서 이미지를 SnapMind로 직접 공유할 수 있습니다. 수신 즉시 앱 내부 저장소에 복사하여 URI 권한 만료를 방지합니다.

Room DB — 로컬 퍼스트 저장소
MemoryItem · Tag · N:M 매핑 테이블로 구성된 Room DB에 이미지 URI, OCR 텍스트, 분류 결과, 태그, 메모를 저장합니다. FTS(Full-Text Search) 인덱스로 빠른 통합 검색을 지원합니다.
