# Bug Tracker

## Status Labels

| Label | Meaning |
| --- | --- |
| Open | Confirmed and not fixed |
| In Progress | Fix is being implemented |
| Fixed | Fix merged |
| Verified | Fix tested |
| Won't Fix | Accepted limitation |

## Severity

| Severity | Meaning |
| --- | --- |
| S0 | Crash, data loss, security/privacy issue |
| S1 | Major feature broken |
| S2 | Important workflow degraded |
| S3 | Minor visual/state issue |
